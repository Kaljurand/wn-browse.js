WordNet browser
===============

Simple jQuery-based WordNet browser.

_Work in progress_


Navigation examples
-------------------

### Synsets only

  - viis -> kanal -> toru
  - ...

### Synsets + hypo-hyper

  - koha -> siga
  - komeet -> kuu -> jaanuar
  - ...
