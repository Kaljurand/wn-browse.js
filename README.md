WordNet browser
===============

Proof-of-concept jQuery-based WordNet browser.

_Work in progress_


Features
--------

  - fast
  - offline
  - simple


Navigation examples
-------------------

### Synsets only

  - viis -> kanal -> toru
  - matk -> kints
  - ...

### Synsets + hypo-hyper

  - koha -> siga
  - komeet -> kuu -> jaanuar
  - ...
